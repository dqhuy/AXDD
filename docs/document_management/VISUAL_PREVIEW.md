### Desktop (â‰¥992px):
- Two-column layout in upload form

- All filters visible in one row

### Tablet (768px - 991px):
- Two-column layout maintained
- Table becomes horizontally scrollable

- Single-column layout in upload form
- Card-based document display instead of table
- Filters stack vertically
- Simplified pagination

## Interaction States

### Hover States:
- Table rows: Light gray background
- Buttons: Darker shade of button color
- Drag zone: Border color changes to blue

### Active States:
- Current page: Bold with different background
- Selected filters: Highlighted
- Active sort column: Indicator arrow

### Loading States:
- Submit button: Spinner icon + "Uploading..." text
- Disabled state until operation completes

### Error States:
- Validation messages in red below inputs
- Alert banner at top of form
- Required fields indicated with red asterisk (*)

### Success States:
- Green alert banner: "Document uploaded successfully"
- Green checkmark icon
- Auto-redirect to index after 2 seconds
